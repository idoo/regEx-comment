# RegEx 
extract comment content from programming languages

## C++
`/^([^\/\/]*)\/\/(.*)$/`

## C#
`/^([^\/\/]*)\/\/(.*)$/`

## Fortan
`/^([^!]*)!(.*)$/`

## Go
`/^([^!]*)!(.*)$/`

## Haskell
`/^([^--]*)--(.*)$/`

## Java
`/^([^\/\/]*)\/\/(.*)$/`

## Javascript
`/(\/\*[\s\S]*?(.*)\*\/)|(\/\/.*)/gm`

## Kotlin
`/^([^\/\/]*)\/\/(.*)$/`

## PHP
`/^([^\/\/]*)\/\/(.*)$/`

## Python
`/^([^#]*)#(.*)$/`

## Ruby
`/^([^#]*)#(.*)$/`

## TypeScript
`/^([^\/\/]*)\/\/(.*)$/`
