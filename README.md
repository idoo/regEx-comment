# RegEx 
extract [currently single line only] comment content from programming languages

## C++
`/^([^\/\/]*)\/\/(.*)$/`

## C#
`/^([^\/\/]*)\/\/(.*)$/`

## Fortan
`/^([^!]*)!(.*)$/`

## Go
`/^([^!]*)!(.*)$/`

## Haskell
`/^([^--]*)--(.*)$/`

## Java
`/^([^\/\/]*)\/\/(.*)$/`

## Javascript
`/^([^\/\/]*)\/\/(.*)$/`

## Kotlin
`/^([^\/\/]*)\/\/(.*)$/`

## PHP
`/^([^\/\/]*)\/\/(.*)$/`

## Python
`/^([^#]*)#(.*)$/`

## Ruby
`/^([^#]*)#(.*)$/`

## TypeScript
`/^([^\/\/]*)\/\/(.*)$/`
